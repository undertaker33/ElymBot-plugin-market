# 02 — 包结构与描述文件

## 目录布局

```
plugin-root/
├── manifest.json              ← 必需：插件身份与元数据
├── android-plugin.json        ← 必需：运行时与配置契约
├── _conf_schema.json          ← 可选：静态配置项定义
├── runtime/
│   ├── bootstrap.js           ← 必需：bootstrap 入口
│   ├── utils.js               ← 可选：自定义模块
│   └── ...
├── schemas/
│   └── settings-schema.json   ← 可选：设置界面 UI schema
└── assets/
    ├── images/
    └── ...                    ← 可选：图片、音频等静态资源
```

## manifest.json 完整字段

```json
{
  "pluginId": "com.example.helloworld",
  "version": "1.0.0",
  "protocolVersion": 2,
  "author": "Your Name",
  "title": "Hello World 示例插件",
  "description": "最小可导入、可执行、可配置的 ElymBot 插件模板",
  "permissions": [],
  "minHostVersion": "0.4.0",
  "maxHostVersion": "",
  "sourceType": "GIT",
  "sourceUrl": "https://github.com/undertaker33/helloworld.git",
  "entrySummary": "演示命令处理、正则匹配、LLM 钩子和配置读取",
  "riskLevel": "LOW"
}
```

| 字段 | 类型 | 必需 | 说明 |
|------|------|------|------|
| `pluginId` | string | ✅ | 全局唯一标识，推荐反向域名 `com.author.plugin-name` |
| `version` | string | ✅ | 语义版本号 |
| `protocolVersion` | number | ✅ | 固定为 `2` |
| `author` | string | ✅ | 作者名 |
| `title` | string | ✅ | 显示名称 |
| `description` | string | ✅ | 一句话描述 |
| `permissions` | string[] | ✅ | 权限列表，目前留空 `[]` |
| `minHostVersion` | string | ✅ | 支持的最低宿主版本 |
| `maxHostVersion` | string | ❌ | 支持的最高宿主版本，留空表示不限 |
| `sourceType` | string | ❌ | `"GIT"` / `"LOCAL_FILE"` 等，标识来源 |
| `sourceUrl` | string | ❌ | 当 `sourceType` 为 `"GIT"` 时指定仓库地址 |
| `entrySummary` | string | ❌ | 功能概要 |
| `riskLevel` | string | ❌ | `"LOW"` / `"MEDIUM"` / `"HIGH"` |

## android-plugin.json 完整字段

```json
{
  "protocolVersion": 2,
  "runtime": {
    "kind": "js_quickjs",
    "bootstrap": "runtime/bootstrap.js",
    "apiVersion": 1
  },
  "config": {
    "staticSchema": "_conf_schema.json",
    "settingsSchema": "schemas/settings-schema.json"
  }
}
```

| 字段 | 说明 |
|------|------|
| `protocolVersion` | 固定 `2` |
| `runtime.kind` | 固定 `"js_quickjs"`——使用 QuickJS 引擎 |
| `runtime.bootstrap` | 入口脚本的**相对路径**（相对于插件根目录） |
| `runtime.apiVersion` | Host API 版本号，当前为 `1` |
| `config.staticSchema` | 指向 `_conf_schema.json` 的相对路径，留空表示无配置 |
| `config.settingsSchema` | 指向设置 UI schema 的相对路径，留空表示无设置界面 |

## 模块系统

- 运行时为 **ES Module**（`import` / `export`）
- 支持从同目录或子目录 `import`，路径相对于 `runtime/`
- 入口文件必须 `export default async function bootstrap(hostApi) { ... }`

```javascript
// runtime/utils.js
export function greet(name) {
  return `你好，${name}！`;
}

// runtime/bootstrap.js
import { greet } from "./utils.js";

export default async function bootstrap(hostApi) {
  hostApi.registerCommandHandler({
    key: "greet-cmd",
    command: "greet",
    aliases: [],
    groupPath: [],
    priority: 0,
    filters: [],
    metadata: {},
    handler(event) {
      event.replyText(greet(event.remainingText || "世界"));
    },
  });
}
```
