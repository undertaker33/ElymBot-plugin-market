# 01 — 快速开始

## 1. 创建插件目录

```
my-plugin/
├── manifest.json           ← 插件身份
├── android-plugin.json     ← 运行时契约
├── _conf_schema.json       ← 配置项定义（可选）
├── runtime/
│   └── bootstrap.js        ← 入口脚本
├── schemas/
│   └── settings-schema.json← 设置界面 schema（可选）
└── assets/                 ← 静态资源（可选）
```

## 2. 最小 manifest.json

```json
{
  "pluginId": "com.yourname.my-plugin",
  "version": "1.0.0",
  "protocolVersion": 2,
  "author": "Your Name",
  "title": "我的插件",
  "description": "一句话描述",
  "permissions": [],
  "minHostVersion": "0.4.0"
}
```

关键字段：
- **pluginId** — 全局唯一，推荐反向域名风格
- **protocolVersion** — 必须是 `2`（v1 已废弃）
- **minHostVersion** — 插件要求的最低宿主版本

## 3. 最小 android-plugin.json

```json
{
  "protocolVersion": 2,
  "runtime": {
    "kind": "js_quickjs",
    "bootstrap": "runtime/bootstrap.js",
    "apiVersion": 1
  },
  "config": {
    "staticSchema": "",
    "settingsSchema": ""
  }
}
```

- **runtime.kind** — 固定 `"js_quickjs"`
- **runtime.bootstrap** — 入口脚本相对路径
- **config.staticSchema** — 指向 `_conf_schema.json`（留空表示无配置）
- **config.settingsSchema** — 指向 `settings-schema.json`（留空表示无设置 UI）

## 4. 最小 bootstrap.js

```javascript
export default async function bootstrap(hostApi) {
  hostApi.log("INFO", "[my-plugin] 加载完成");

  hostApi.registerCommandHandler({
    key: "my-plugin-ping",
    command: "ping",
    aliases: [],
    groupPath: [],
    priority: 0,
    filters: [],
    metadata: {},
    handler(event) {
      event.replyText("pong!");
    },
  });
}
```

要点：
1. 必须使用 `export default` 导出一个 **async function**
2. 函数名必须是 `bootstrap`
3. 接收唯一参数 `hostApi`，所有注册和工具方法都在上面

## 5. 导入与运行

1. 将整个插件目录复制到 ElymBot 的插件目录下
2. 在 ElymBot 设置 → 插件管理中刷新
3. 找到你的插件并启用
4. 在聊天中发送 `/ping`，应收到 `pong!`
